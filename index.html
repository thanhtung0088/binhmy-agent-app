Agent Phân tích & Dự báo

chưa chạy lần nào

Phân tích xu hướng dữ liệu các kỳ đã lưu, đưa ra nhận định và khuyến nghị cho lãnh đạo xã.

Kết nối hệ thống

App gọi AI thông qua hàm serverless /api/gemini trên Vercel (không lộ API key ra trình duyệt), và lưu dữ liệu qua Firebase Firestore. Điền thông tin bên dưới nếu cần đổi project Firestore đang dùng (mặc định app dùng chung project với Cổng thông tin UBND Xã Bình Mỹ).
